pipes

pipes(Uppercase, lowercase. The Symbol of pipe is to represent |)

  • Pipes (|) in Angular are used to transform the data before displaying it in a browser. Angular provides a lot of built-in pipes to translate the data before displaying it into the browser and as we know, Angular lets us extend its feature, we can even create custom pipes in Angular.
  • Pipes which we can apply to Strings Pipes which we can apply to Numbers Pipes which we can apply to Date

Original String - {{ name }}

uppercase pipe - {{ name | uppercase }}

lowercase pipe - {{ name | lowercase }}

slice pipe - {{ name | slice: 3: 9}}

Original Number - {{ number }}

number number - {{ number | number: '1.3-3' }}

currency number - {{ number | currency:'USD':true }}

currency number - {{ number | currency:'EUR':true}}
